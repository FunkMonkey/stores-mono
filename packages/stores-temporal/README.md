# @crikey/stores-temporal

Provides time-based store transformations such as {@link debounce} and {@link throttle}.  

## Installation

```bash
# pnpm
$ pnpm add @crikey/stores-temporal

# npm
$ npm add @crikey/stores-temporal

# yarn
$ yarn add @crikey/stores-temporal
```
## Usage

See:
* {@link debounce}
* {@link throttle}
