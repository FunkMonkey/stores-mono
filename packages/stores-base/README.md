# @crikey/stores-base

Core package used for the implementation of various [Svelte](https://svelte.dev/) compatible stores.

## Installation

```bash
# pnpm
$ pnpm add @crikey/stores-base

# npm
$ npm add @crikey/stores-base

# yarn
$ yarn add @crikey/stores-base
```
